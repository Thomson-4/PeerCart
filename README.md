## 🧠 Overview
**PeerCart** is a smart, AI-powered peer-to-peer marketplace that enables users to **buy, sell, and rent items safely and efficiently** within a trusted ecosystem.

Unlike traditional platforms like OLX, PeerCart focuses on **trust, speed, and intelligent matching**.

---

## 🎯 Problem
- Lack of trust (fake users & scams)  
- Unorganized platforms (WhatsApp, social media)  
- Time-consuming search for buyers/sellers  
- No proper renting system  

---

## 💡 Solution
PeerCart provides a **verified, intelligent platform** where users can:
- Buy, sell, and rent items  
- Connect with trusted users  
- Complete transactions quickly and safely  

---

## 🔑 Features

- 🛒 **Buy • Sell • Rent**
- 🔐 **User Verification System**
